Bitcoin/Litecoin Payments with WP e-Commerce plugin for WordPress.

== Description ==

If your online store uses e-Commerce platform, you can start accepting Bitcoin/Litecoin payments with this plugin for WordPress e-Commerce.
All you need is <a href="https://www.btcmerch.com/auth/signup" rel="nofollow">SignUp at BTCMerch.com</a> and specify your Bitcoin/Litecoin address.
<br /><br />There is a <a href="https://www.btcmerch.com/help/sandbox" rel="nofollow">Sandbox</a> for testing the integration of bitcoin/litecoin to your online store. 
This allows application developers or testers to experiment without having to use real bitcoins/litecoins.

== Installation ==

1. Sign Up at <a href=https://www.btcmerch.com/ rel="nofollow">Btcmerch.com</a> for free
2. Fill in the fields 'Your Bitcoin address' and/or 'Your Litecoin address'
2. Upload `wp-e-commerce` folder to the `/wp-content/plugins/` directory
3. Open the tab 'Payments' in Store Settings and check the box 'Bitcoins/Litecoins'
4. Click 'Settings' of 'Bitcoins/Litecoins' payment gateway
5. Copy 'Your callback URL' from the plugin settings in Wordpress into your BTCMerch profile 'Callback URL'
6. Copy 'Merchant ID' and 'API Secret Word' from BTCMerch profile to the Wordpress payment method settings
7. Save all changes

== Frequently Asked Questions ==

= What is BTCMerch? =

<a href=https://www.btcmerch.com/ rel="nofollow">BTCMerch</a> is an electronic payment processing system for bitcoin and litecoin currencies. We enable online merchants to accept bitcoins and litecoins, as a form of payment, just as they accept payments from credit cards or Paypal.

= How much does it cost to use BTCMerch? =

There is no set up or monthly fees. There is a transaction fee of just 0.5%. The minimal transaction fee is 0.002 BTC or 0.06 LTC.

= What is the minimal amount to pay with BTCMerch? =

The minimal amount is 0.01 BTC or 0.4 LTC.

= What currencies can I sell goods with? =

Merchants can use BTC (bitcoins), LTC (litecoins) or the currencies from the list below. 
If the amounts in currencies are different from BTC and LTC, the currency will be converted to BTC and LTC 
at the market rates.
Supported currencies are: USD, EUR, JPY, CAD, GBP, CHF, RUB, AUD, SEK, DKK, HKD, PLN, CNY, SGD, THB, NZD, NOK.
